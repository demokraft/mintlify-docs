## Overview of Objects

Objects serve as ready-to-use mockup devices such as laptops, desktops, frames, glass panels and more that function as display elements within your video scenes. They enable you to showcase your product visually by displaying an image or video content. Each object supports animation and visual effects, enhancing viewer engagement and providing a dynamic presentation.

## Accessing the Objects Section

To incorporate objects into your video project, open your project within the Studio editor. Navigate to the **Objects** section located in the navigation panel, which provides access to the available mockup devices you can add.

## Adding Objects to the Scene

Choose the desired object type from the available options: laptop, desktop, frame, glass panel or others. Position the player marker within the scene at the location where you want to insert the object. Adding the object at this point immediately renders it within your scene for further configuration.

## Configuring Object Animation and Effects

Once the object is added, use the canvas to review its animation. Adjust object settings through the configuration panel, which includes options for animation duration, motion effects, and visual enhancements. Use the preview feature to observe how the object and its animations integrate within the scene.

## Uploading Product Screen Content

If an object does not yet contain any uploaded product screen, it will display as empty within the scene. To proceed, upload the product screen content — this can be an image or video — directly to the object to populate its display.

## Understanding Product Screen Content

Product screen content refers to the visual media, such as images or video clips, representing your product or interface. This content is uploaded and displayed through the object, allowing your product to be visually presented within the mockup.

## Positioning and Animation Settings

Drag and drop the object within the scene to adjust its placement as needed. You can configure both entry and exit animations for the object, specify slide directions, and modify other animation settings to refine how the object appears and disappears within the video.

## Previewing Object Appearance

After configuring the object and its animations, utilize the preview function to verify the final appearance and behavior of the object within your scene before saving changes. This ensures the visual effect meets your expectations prior to final application.

## Saving and Rendering Object Animations

Applying configuration settings saves the changes directly to your project. The system then initiates rendering for the object animation, which is incorporated into the final exported video, ensuring the object appears as configured in the finished product.
To remove an object, click the layer element of the object so that its config panel appears in navigation panel. You can then delete the object by clicking Delete button in the object config or by selecting Delete in right click option on object layer elment.
