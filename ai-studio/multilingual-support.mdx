From one base recording, AI Studio translates your approved script into any supported language, selects a native-speaker AI voice for that language, and generates a fully synchronised voiceover — producing a standalone, independently editable project for each target language. No re-recording, no external translation service, and no separate production run. Your visuals stay exactly the same; only the narration and script change, keeping the viewer experience consistent across every language version.

## Review Workflow

<Steps>
  <Step title="Open the Languages panel">
    Open any completed project (script approved, voiceover generated) and open the **Languages** panel in the editor.
  </Step>
  <Step title="Select a target language">
    Click **Add Language**, then select your target language from the list. AI Studio queues a translation job for each language you add.
  </Step>
  <Step title="Find your translation projects">
    Each translation appears as a separate project in your project list, named with the original project title and the target language code (e.g. _"Product Demo — FR"_).
  </Step>
  <Step title="Review and edit">
    Open a translation project to review the translated script. Edit any segment inline — changes trigger a voiceover re-generation for that segment only, leaving all other segments untouched.
  </Step>
  <Step title="Export or send to AI Hub">
    Export each language version independently using the same options as your base project, or click **Send to Hub** to add all versions to your AI Hub content library at once.
  </Step>
</Steps>

## Regional Variants

Regional variants — such as Spanish (Spain) vs Spanish (Latin America), or Portuguese (Brazil) vs Portuguese (Portugal) — are selected automatically based on the target language you choose. The appropriate regional voice is applied by default and can be overridden in the Voiceover panel of the translation project.

## Multilingual Demos in AI Hub

In AI Hub, combine all your language versions into a single demo session and enable the **language picker**. Viewers see a language selector at the start of the experience and can switch languages at any point — without leaving the demo.

<Tip>
  Enable the language picker in AI Hub for international prospects — they can switch to their preferred language without leaving the demo, removing friction for non-English audiences and increasing engagement for regional campaigns.
</Tip>
